# 나의 영양제 루틴 (Modo)

> 매일 챙겨먹는 영양제를 시간대별로 체크하는 모바일 친화 웹 앱

홈 화면에 설치하면 네이티브 앱처럼 열리고, 인터넷이 끊겨도 동작합니다. 자정이 지나면 어제의 체크가 자동으로 초기화되어 매일 새로 시작할 수 있습니다.

## 주요 기능

### ✅ 시간대별 복용 스케줄
하루를 5개 시간 블록으로 나눠 영양제를 정리합니다.

| 시간대 | 조건 | 등록된 영양제 |
|--------|------|---------------|
| 기상 직후 | 공복 | 🌿 매스틱검 |
| 아침 식사 30분 전 | 공복 | 🦠 유산균 |
| 아침 식사 중 / 후 | 식사와 함께 | 💊 종합비타민 |
| 저녁 식사 중 / 후 | 식사와 함께 | 💊 종합비타민, 🐟 오메가3, 🩸 철분제 |
| 취침 전 | 저녁 식후 2시간 후 | 🌙 마그네슘 글리시네이트 |

각 카드에는 복용량(예: "2캡슐 · 1,000mg")과 흡수에 좋은 복용 팁이 함께 표시됩니다.

### ✅ 체크 인터랙션
- 카드 왼쪽 체크박스를 누르면 복용 완료 표시
- 체크된 카드는 흐려지고 영양제 이름에 취소선
- 잘못 체크하면 다시 눌러 해제 가능

### ✅ 자동 초기화 + 수동 초기화
- 자정이 지나면 어제 체크는 모두 사라지고 새로운 하루 시작
- "오늘 체크 초기화" 버튼으로 즉시 모두 해제 가능

### ✅ 상태 유지
앱을 닫고 다시 열어도, 새로고침해도 오늘 체크한 상태는 그대로 유지됩니다 (브라우저 `localStorage` 사용).

### ✅ 흡수 방해 조합 안내
하단의 "주의사항" 영역에서 함께 먹으면 안 되는 조합을 알려줍니다.
- 철분 + 마그네슘은 최소 2시간 간격
- 철분 + 커피·녹차·홍차는 전후 1시간 피하기 (타닌이 흡수 방해)
- 유산균 + 항생제는 최소 2시간 간격
- 철분은 비타민C 음료와 함께 → 흡수율 30% 향상

### ✅ 오프라인 동작 (PWA)
- 처음 한 번 열어두면 네트워크가 끊겨도 정상 동작
- 모바일 홈 화면에 추가하면 주소창 없는 전체화면 앱으로 실행
- iOS / Android 모두 지원

## 설치 방법

### iPhone (Safari)
1. Safari로 앱 주소를 연다
2. 하단 공유 버튼 (↑) → **"홈 화면에 추가"** 선택
3. 홈 화면의 영양제 아이콘으로 실행

### Android (Chrome)
1. Chrome으로 앱 주소를 연다
2. 우측 상단 메뉴(⋮) → **"홈 화면에 추가"** 또는 자동으로 뜨는 설치 안내 수락
3. 홈 화면의 영양제 아이콘으로 실행

### PC 브라우저
- 그냥 즐겨찾기에 추가해서 사용하거나, Chrome/Edge의 "앱으로 설치" 옵션을 사용할 수 있습니다.

---

## 사용 팁

- **아침에 일어나자마자** 앱을 열어 "기상 직후" 영양제를 챙기는 것을 루틴으로 만드세요
- 식사 직전에 미리 영양제를 꺼내두고, 먹은 후 바로 체크하면 빠뜨리지 않습니다
- 자정 직후에 앱을 열면 자동 초기화 직전 상태일 수 있으니, 아침에 다시 한 번 확인하세요
- 이상 반응이 느껴지면 즉시 중단하고 의사·약사와 상담하세요
